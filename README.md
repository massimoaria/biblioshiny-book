This is the official companion repository for the book **"A Primer on Science Mapping Analysis with Biblioshiny"**. This repository provides the digital resources, datasets, and supplementary materials required to follow the workflows described in the manual.

---

## 📖 Overview

This repository is designed to bridge the gap between theory and practice. Readers will find all the necessary files to replicate the bibliometric analyses presented in the book, ensuring a hands-on learning experience with the `bibliometrix` R-package and its web interface, **Biblioshiny**.

## 📁 Repository Structure

The contents are organized as follows:

* **/datasets**: Contains raw bibliographic data (Plain Text and BibTeX formats) exported from Web of Science (WoS) and Scopus. These files are used in the step-by-step tutorials.
* **/supplementary-materials**: High-resolution versions of the scientific citation network models, figures, and extended methodological notes.
* **/scripts**: R scripts for advanced users looking to customize their bibliometric workflows beyond the GUI.
* **/exercises**: Practical tasks and self-assessment files to test your science mapping skills.

## 🚀 How to Use These Resources

1.  **Clone the Repository**: 
    ```bash
    git clone [https://github.com/massimoaria/biblioshiny-book.git](https://github.com/massimoaria/biblioshiny-book.git)
    ```
2.  **Locate the Datasets**: Access the `datasets/` folder to find the ZIP archives mentioned in **Chapter 3 **.
3.  **Follow the Manual**: The files are named according to the corresponding chapters and case studies in the book.
